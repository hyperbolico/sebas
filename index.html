1. SUCESIONES CUADRÁTICAS Identificar tipo Sucesión simple: diferencia constante
Sucesión cuadrática: segunda diferencia constante		 Ejemplo: 3, 8, 15, 24, 35...
• 1ras diferencias: +5, +7, +9, +11 (NO ctes)
• 2das diferencias: +2, +2, +2 (SÍ ctes)
✓ Es CUADRÁTICA
Fórmula general an² + bn + c
• 2a = 2da diferencia
• 3a+b = 1ra diferencia inicial
• a+b+c = 1er término		 Para 3,8,15,24,35:
• 2a = 2 → a=1
• 3(1)+b=5 → b=2
• 1+2+c=3 → c=0
Fórmula: n² + 2n
Calcular términos Sustituir n en la fórmula n² + 2n
• T₁ = 1²+2(1)=3 ✓
• T₅ = 5²+2(5)=25+10=35 ✓
• T₁₀ = 10²+2(10)=100+20=120
2. PROPIEDADES DE EXPONENTES Multiplicación aᵐ · aⁿ = aᵐ⁺ⁿ x⁵ · x³ = x⁵⁺³ = x⁸
Explicación: Misma base (x), sumamos exponentes 5+3=8
División aᵐ / aⁿ = aᵐ⁻ⁿ y⁷ / y² = y⁷⁻² = y⁵
Explicación: Restamos exponentes 7-2=5
Potencia de potencia (aᵐ)ⁿ = aᵐ·ⁿ (z⁴)³ = z⁴ˣ³ = z¹²
Explicación: Multiplicamos exponentes 4×3=12
2. DESIGUALDADES Despejes y cambios de signo ⚠️ REGLA CLAVE: Al multiplicar/dividir por NEGATIVO, el signo se INVIERTE -3x < 12
Dividir entre -3: (-3x)/-3 < 12/-3 → ¡CAMBIA! → x > -4
Problemas Traducir enunciado a desigualdad "Gastar menos de $200 con $25 c/u y $45 extra"
25x + 45 < 200
25x < 155
x < 6.2 → x ≤ 6 bolsas
3. POLÍGONOS Suma de ángulos internos (n - 2) × 180°
n = número de lados		 Hexágono (n=6):
(6-2)×180° = 4×180° = 720°
Número de diagonales D = n(n - 3)/2 Octágono (n=8):
8(8-3)/2 = 8×5/2 = 40/2 = 20 diagonales
4. FIGURAS A ESCALA Aumento Multiplicar medidas por factor > 1 Rectángulo 3×5 cm, escala 4:
3×4=12 cm, 5×4=20 cm → 12×20 cm
Disminución Multiplicar medidas por factor 0 < k < 1 Rectángulo 3×5 cm, escala 0.5:
3×0.5=1.5 cm, 5×0.5=2.5 cm → 1.5×2.5 cm
5. CONCEPTOS TEÓRICOS Igualdad (=) Dos expresiones tienen exactamente el MISMO valor 3x + 2 = 11
Solo es cierto cuando x=3: 3(3)+2=9+2=11 ✓
Desigualdad (>,<,≥,≤) Las expresiones tienen diferente valor y hay un orden 3x + 2 > 11
Es cierto para x>3: x=4 → 14>11 ✓; x=3.5 → 12.5>11 ✓
